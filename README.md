# 🎵 Discord Music Bot

Um bot de música para Discord feito com **discord.py**, com sistema completo de fila, playlists e controle de reprodução.

---

## 🚀 Funcionalidades

- 🔎 Buscar músicas pelo nome (YouTube)
- 🎶 Sistema de fila
- 🔀 Embaralhar músicas
- ⏮ Música anterior
- ⏭ Avançar / retroceder tempo
- 🔁 Repetição infinita
- 🧹 Limpar fila
- 📂 Playlists pessoais
- ⏹ Parar música
- 🔌 Desconectar do canal de voz

---

## 🛠️ Tecnologias utilizadas

- Python
- discord.py
- yt-dlp
- FFmpeg
